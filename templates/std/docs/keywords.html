Ключевые слова
 
на главную | содержание

Ключевое слово представляет собой произвольную последовательность из латинских букв, цифр и знаков пунктуации. Задача может водержать произвольное количество ключевых слов. Система не накладывает ограничений на структуру и допустимые сочетания слов.

По соглашению, ключевые слова состоят из последовательностей малых латинских букв, разделённых точками. Первая часть отвечает за вид классификации, последующие -- за постепенное уточнение раздела. Некоторые слова, значение которых не очевидно из названия, подробно описаны ниже:

  • alg -- классификация по применяемым алгоритмам и методам
  • cmp -- классификация по сложности решения
    • cmp.trivial -- задача решается тривиальным образом, и не требует от решающего самостоятельного мышления. Такие задачи не включаются в олимпиады либо включаются в качестве "утешительных".
    • cmp.open-problem -- жюри неизвестно полное решение задачи. Эталонное решение отсутствует или работает не на всех тестах, либо существуют известные контрпримеры, не включенные в пакет.
  • o -- классификация по алгоритмической сложности
    • linear -- O(n)
    • poly -- O(nq)
    • subpoly -- сложность между линейной и полиномиальной, обычно O(n log n)
    • exp -- O(2n)
