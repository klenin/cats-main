Описание

Просмотреть турнирную таблицу можно, выбрав пункт меню "результаты". Таблица автоматически обновляется каждые 30 секунд. Таблица содержит следующие колонки:

  • Порядковый номер
  • Название команды
  • Результаты по задачам -- колонки, озаглавленные латинскими буквами -- кодами задач. Наведение мыши на код задачи позволяет просмотреть название задачи, а нажатие -- просмотреть текст задачи. Фактическое содержание колонок определяется правилами соревнований.
  • Сумма баллов/количество решённых задач/время -- в зависимости от правил соревнований.
  • Место, занятое участником (может отличаться от порядкового номера в случае разделения мест).
  • Статус участника

Статус участников

Статус участника состоит из следующих признаков:

  • remote -- участник находится вне области физического контроля организаторов соревнования
  • ooc -- участие вне конкурса, чаще всего remote подразумевает ooc
  • virtual -- участие со сдвигом по времени, virtual всегда подразумевает ooc

Фильтрация

С помощью ссылок "спрятать ooc-команды" и "спрятать виртуальные команды" можно выбирать подмножество участников, выводимых в турнирной таблице. Официальне результаты турнира выводятся в первом из этих режимов.
