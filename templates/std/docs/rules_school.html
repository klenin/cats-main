Оценка результатов

Задачи проверяются после окончания тура на наборе тестов, одинаковом для всех участников. За каждый тест начисляется определённое количество баллов. Максимально возможный балл за задачу указан в условии задачи. После проверки участникам сообщаются результаты каждого теста.

Во время тура решения сдаются в тестирующую систему и проверяются на наборе тестов, приведённых в условии. Решения, не прошедшие ни одного теста из условия, не проверяются по окончании тура и получают оценку 0 баллов. Решения, прошедшие хотя бы один тест из условия, гарантированно набирают не менее 1 балла.

Время отправки решения и количество попыток не учитываются, однако общее количество попыток не может превышать 20 на одну задачу. Для проверки используется последняя сделанная попытка.

Участники упорядочиваются по убыванию суммы баллов, набранной по всем задачам. Участники, набравшие одинаковое количество баллов, разделяют общее место.

Ограничения видимости задач

В случае, если при подготовке задач использовались материалы методической комиссии Всероссийской олимпиады по информатике, видимость задач ограничивается согласно требованиям комиссии. Задачи таких турниров видны только участникам, лично присутствующим на олимпиаде, и не видны при дистанционной регистрации.

Задачи могут быть опубликованы после окончания всех региональных олимпиад. Фактически жюри использует дату начала ближайшей Всероссийской олимпиады.
