Просмотр задач и отправка решений
 
на главную | содержание

Для просмотра списка задач текущего турнира сделует выбрать пункт меню "задачи". Прочесть отдельную задачу можно, нажав на её название. Кроме того, пункт локального меню "задачи для печати" выдаёт текст всех задач в виде, пригодном для вывода на принтер.

Для отправки решения на проверку необходимо:

  1. Убедиться, что вы являетесь участником выбранного турниря
  2. Выбрать задачу из списка нажатием на радио-кнопку слева от названия.
  3. Ввести или выбрать имя файла с решением.
  4. Если расширение файла не совпадает с расширением, указанным в списке компиляторов, выбрать среду разработки.
  5. Нажать кнопку "отправить".

Ключевые слова

Для поиска задач в системе можно использовать ключевые слова. Для этого нужно перейти к пункту меню "ключевые слова" и выбрать нужное слово из списка. Будут выданы все задачи, в описании которых присутствует выбранное слово. Более подробное описание назначения некоторых ключевых слов можно просмотреть здесь.
