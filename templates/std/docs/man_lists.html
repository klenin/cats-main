Турниры

  • Название
  • Начало
  • Завершение
  • Примечание:
    • официальный -- во время проведения офицальных турниров запрещена отправка заданий в остальные турниры
    • регистрация закрыта -- турнир ещё не начался, и зарегистрироваться в нём пока нельзя
    • школьные правила -- задачи оцениваются по правилам школьных олимпиад

Задачи

  • Задача -- содержит название задачи, перед которым может идти латинская буква -- код задачи в текущем турнире. После названия задачи может идти ссылка "[пакет]", позволяющая скачать пакет с тестами и эталонными решениями в формате CATS.
  • accepted - количество успешных попыток решения данной задачи, сделанных текущим пользователем (для Training Session -- всеми пользователями).
