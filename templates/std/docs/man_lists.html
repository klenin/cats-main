Задачи

  • Задача -- содержит название задачи, перед которым может идти латинская буква -- код задачи в текущем турнире. После названия задачи модет идти ссылка "[пакет]", позволяющая скачать пакет с тестами и эталонными решениями в формате CATS.
  • accepted - количество успешных попыток решения данной задачи, сделанных текущим пользователем (для Training Session -- всеми пользователями).
